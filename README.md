# Habitus – Habit Tracker App Landing Page

Это лендинг страница для приложения, отслеживающего привычки. 

Проект не предназначен для коммерческого использования или промышленного развёртывания. Он демонстрирует базовые концепции веб‑разработки и может служить шаблоном для начинающих.

## Этот проект был создан в учебных целях для:
* закрепления навыков работы с **HTML5**, **CSS3**, **Bootstrap 5** и **JavaScript (ES6+)**;
* освоения принципов построения интерактивных пользовательских интерфейсов;
* отработки структуры статического веб‑приложения.

## Применяются различные технология:  

Проект построен на классическом стеке веб‑разработки:
* **HTML5** — структура страницы;
* **CSS3** — стилизация и **адаптивный дизайн** (в т. ч. ручная адаптация под мобильные устройства — исходный макет был только для ПК);
* **Bootstrap 5** — готовые компоненты (карточки, кнопки, сетка), адаптивность и JavaScript‑плагины;
* **JavaScript (ES6+)** — интерактивность, логика приложения.
